# arp_spoofer
A sophisticated tool designed to simulate Man-in-the-Middle (MitM) attacks through ARP cache poisoning. It allows security researchers to intercept traffic between a victim and a gateway to study network vulnerabilities.

# Key Features:

  Dual-target poisoning (Victim and Gateway) for full traffic interception.

  Intelligent MAC address discovery and error handling.

  Automatic network restoration feature to ensure connectivity after testing.
